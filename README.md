# Aviad Plugin Marketplace

## Install

```
claude plugin install AviadP/aviad-plugin-marketplace
```

## Plugins

| Plugin | Description |
|--------|-------------|
| **aviad-dev-toolkit** | Architecture planning, code quality review, security review, verification, refactoring, test planning, memory, and shipping workflows |
| **minihive** | Multi-agent orchestrator — PM plans, agents build in parallel, test, review, commit |
